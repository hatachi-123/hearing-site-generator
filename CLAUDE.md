# Web制作 共通ルール

## 必読ファイル
- **WEB_DESIGN_TECHNIQUES.md** — デザイン技法リファレンス。プロンプトに `TECH:①③⑤` のようなIDが含まれている場合、このファイルの技法カタログに従って実装すること。

## 技術スタック
- HTML / CSS / JavaScript（フレームワークなし）
- フォント：Google Fonts
- フォーム：Formspree
- アニメ：Intersection Observer API
- デプロイ：Cloudflare Pages

## デザイン共通ルール（必ず守ること）
- 角丸：border-radius 12px以上
- セクション余白：padding 80px以上
- アニメ：opacity 0→1・translateY 40px→0・duration 0.6s
- セクション区切り：SVGウェーブを使う
- モバイルファースト：CSSはスマホ（375px）を基準に書き、min-widthメディアクエリでタブレット（768px）・PC（1280px）に拡張する
- フォントペアリング：見出し用と本文用の2種類を必ず使う
- カラーは60:30:10の法則で構成する

## CSS変数（style.cssの冒頭に必ず定義）
プロンプトで指定されたカラーとフォントを以下の変数名で必ず定義すること。

```css
:root {
  --color-base:      （プロンプトのベースカラー）;
  --color-main:      （プロンプトのメインカラー）;
  --color-accent:    （プロンプトのアクセントカラー）;
  --color-text:      #2C2C2C;
  --font-heading:    （プロンプトの見出しフォント）;
  --font-body:       （プロンプトの本文フォント）;
  --radius:          12px;
  --section-padding: 80px;
}
```

## アイコンルール
- 絵文字・アイコン絵文字の使用禁止（☕🍰🌿等）
- アイコンが必要な場合はSVGで実装する
- SVGアイコンはインラインで直接HTMLに書く
- 外部アイコンフォント（Font Awesome等）は使わない

## 禁止事項（全案件共通・絶対に守ること）
- Interフォント禁止
- 紫グラデーション禁止
- 白一色の背景禁止
- Bootstrap・Tailwind等CSSフレームワーク禁止
- jQuery禁止（バニラJSで書く）
- インラインCSS禁止
- console.log禁止
- 絵文字・アイコン絵文字の使用禁止（☕🍰🌿等）

## HTMLルール（全案件共通）
- セマンティックタグを使う（section・article・nav等）
- 全画像にalt属性をつける
- 全ページにtitleとmeta descriptionを設定する
- Google FontsはHTMLのheadで読み込む
- CSSはlinkタグで読み込む
- JSはbodyの閉じタグ直前で読み込む
- 外部リンクにtarget="_blank"とrel="noopener"をつける

## AI生成感を消すルール（必ず守ること）

### レイアウト
- 全セクションを同じレイアウトにしない（3カラム均等グリッドの繰り返し禁止）
- コンテンツの優先度に応じてサイズに差をつける（例：推しメニューは大きく、他は小さく）
- 非対称レイアウトを最低1箇所入れる（例：左テキスト+右写真の5:6比率など）
- 「セクションタイトル→グリッド→ボタン」の同一パターンを3回以上連続させない

### セクション区切り
- SVGウェーブは最大2箇所まで（場面転換が大きい箇所のみ）
- それ以外の区切りは余白・背景色の切り替え・細線などを使い分ける

### コピーライティング
- 固有名詞を必ず入れる（産地名・ブランド名・地名・素材名など）
- 「厳選した」「贅沢な」「こだわりの」だけで終わる抽象的な表現を避ける
- 数字を入れる（「毎朝7時から」「48時間以内」「1日15食限定」など）
- ストーリー性のあるセクションを1つ入れる（店主の経歴・開業の経緯など）

### 写真・ビジュアル
- 写真なしでサイトを完成としない（最低限ヒーロー・メニュー・雰囲気の3枚）
- SVGアイコンだけのカード画像を避ける（写真 or 写真プレースホルダーを使う）
- パーティクルアニメーションなどの「それっぽい演出」より、写真のスライドやパララックスを優先する

### CTA（行動喚起）
- CTAボタンの過剰配置を避ける（同じCTAは1ページ最大2箇所まで）
- ヒーロー・CTA専用セクション・フッターすべてに同じ予約ボタンを置かない
