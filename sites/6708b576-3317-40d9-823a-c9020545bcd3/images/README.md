## 画像撮影のポイント

### 共通要件
- 明るく自然な照明
- 高解像度（最低でも指定サイズの2倍）
- JPEG形式（品質80-90%）
- 清潔感・プロフェッショナル感を重視

### 色調・雰囲気
- サイトのカラーパレットと調和
- ベース：#F8F9FA（清潔感のある薄いグレー）
- メイン：#2E7D8A（信頼感のあるティールブルー）
- アクセント：#E8A547（温かみのあるオレンジ）

### 避けるべき要素
- 過度に派手な演出
- 人工的すぎる照明
- プライバシーを侵害する可能性のある内容
- ブランディングと合わない雰囲気

## 代替案（撮影が困難な場合）

自前撮影が困難な場合は、以下の無料素材サイトから類似画像を取得可能です：

- **Unsplash**: https://unsplash.com/
