# 必要な画像ファイル一覧

羽淵民泊Webサイトで使用する画像ファイルの仕様です。

## 1. ヒーロー動画
- **ファイル名**: `hero-video.mp4`
- **推奨解像度**: 1920x1080 (Full HD)
- **長さ**: 30-60秒（ループ再生）
- **内容**: 宿泊施設の外観・内装の美しい映像
- **フォーマット**: MP4 (H.264コーデック)
- **ファイルサイズ**: 20MB以下

## 2. 客室写真（Sticky Scroll用）
### リビングルーム
- **ファイル名**: `living-room.jpg`
- **推奨解像度**: 1200x800
- **内容**: カッシーナ社製ソファとアンティーク調テーブルが映る高級リビング

### ベッドルーム
- **ファイル名**: `bedroom.jpg`
- **推奨解像度**: 1200x800
- **内容**: シモンズ社製マットレスと高級リネンが映る寝室

### キッチン
- **ファイル名**: `kitchen.jpg`
- **推奨解像度**: 1200x800
- **内容**: ガガナウ社製調理器具とノリタケ食器が映るキッチン

### バスルーム
- **ファイル名**: `bathroom.jpg`
- **推奨解像度**: 1200x800
- **内容**: ジャグジー付きバスタブと高級アメニティが映る浴室

## 3. ヒーロー背景画像（動画の代替用）
- **ファイル名**: `hero-bg.jpg`
- **推奨解像度**: 1920x1080
- **内容**: 宿泊施設の美しい外観または内装のメイン画像
- **フォーマット**: JPEG (高画質)

## 画像の品質要件

### 解像度・品質
- 最低解像度: 1200px（横幅）
- JPEG品質: 85-90%
- WebP対応推奨（パフォーマンス向上のため）

### 撮影・編集指針
- **照明**: 自然光を活かした明るい印象
- **色調**: 温かみのある色温度（3000-4000K）
- **構図**: 調度品の高級感が伝わる角度
- **後処理**:
  - コントラスト適度に調整
  - 彩度をやや抑えて上品な印象
  - シャープネス適度に適用

### コンテンツガイドライン
1. **高級感の演出**
   - ブランド家具を明確に識別できる角度
   - 素材感（レザー、木材、ファブリック）が伝わる質感
   - 清潔感と整理整頓された空間

2. **ストーリー性**
   - 実際に宿泊している情景を想像できる構図
   - 時間の流れを感じられるライティング
   - 季節感や時間帯の演出

3. **ブランドイメージ**
   - ダークトーンをベースとした落ち着いた配色
   - ゴールドアクセント（調度品、装飾品）を効果的に配置
   - ミニマルで洗練された印象

## 画像最適化

### ファイル形式
- **JPEG**: 写真画像（グラデーション多い）
- **WebP**: モダンブラウザ対応、軽量化
- **PNG**: アイコン、透過画像（必要に応じて）

### レスポンシブ対応
各画像に対して以下のサイズを用意することを推奨：
- **Desktop**: 1200px幅
- **Tablet**: 800px幅
- **Mobile**: 400px幅

### 読み込み最適化
- 重要な画像（hero, living-room）は先読み
- その他の画像はlazy loading対応
- WebPフォーマットでのファイルサイズ削減

## 代替テキスト（alt属性）例

```html
<img src="living-room.jpg" alt="リビングルーム - デザイナーズソファとアンティーク調テーブル">
<img src="bedroom.jpg" alt="ベッドルーム - 高級マットレスと上質なリネン類">
<img src="kitchen.jpg" alt="キッチン - 高級調理器具と厳選された食器類">
<img src="bathroom.jpg" alt="バスルーム - ジャグジー付きバスと高級アメニティ">
```

## 著作権・ライセンス

- 全ての画像は商用利用可能な素材を使用
- ブランド品（カッシーナ、ガガナウ等）の商標に配慮
- モデルが写っている場合は肖像権の使用許可を取得
- 建物・インテリアの撮影許可を事前に取得
