## 画像撮影のポイント

### 品質要件
- 解像度：最低でも指定サイズ以上
- フォーマット：JPEG（品質85%以上）またはWebP
- 色味：自然な色合い、彩度は控えめに調整
- 明るさ：明るく清潔感のある印象

### 撮影のコツ
1. **商品写真**
   - 自然光での撮影を推奨
   - 背景は白またはナチュラルな木目
   - 商品の新鮮さが伝わるようにアップで撮影
   - 水滴や艶感を活かす

2. **店舗写真**
   - 清潔感と活気が伝わるように
   - お客様の顔は映らないよう配慮
   - 商品の豊富さがわかるアングル
   - 朝の入荷直後の新鮮な状態で撮影

3. **人物写真**
   - 自然な笑顔
   - 作業着やエプロンでプロフェッショナル感
   - 背景は店舗や野菜が少し見える程度
